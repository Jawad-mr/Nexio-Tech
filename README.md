# Nexio Tech

Operating Systems for Real Businesses.

Nexio Tech builds point-of-sale, membership management, and AI software for bakeries, gyms, restaurants, and hotels.

## Tech Stack
* HTML5
* Vanilla CSS (Vercel-inspired theme)
* Javascript (Interactive Canvas Dot Grid)

## Projects
* **NX-01 Bakery POS**: Billing, inventory, and sales tracking.
* **NX-02 Gym Management**: Memberships, attendance, and trainer scheduling.
* **NX-03 Restaurant POS**: Table management, kitchen displays, and billing.
* **NX-04 Hotel Management**: Bookings, housekeeping, and front desk ops.
* **NX-05 AI Chatbot**: Conversational customer support layer.
* **NX-06 E-Books Platform**: Storefront for digital publishing.
